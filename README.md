# War Thunder: Best Settings 🎮⚙️

[![GitHub last commit](https://img.shields.io/github/last-commit/NikitaKasovsky/war-thunder-best-settings)](https://github.com/NikitaKasovsky/war-thunder-best-settings/commits/main)

Готовый файл настроек управления для игры War Thunder, который поможет вам быстрее освоиться в небе и на земле. Настройки оптимизированы для удобства и эффективности.

Прошу заметить, что настройки сформированы для мышки с двумя боковыми клавишами (MouseButton4 и MouseButton5) и клавиатуры формата 75%.

## ✨ Что внутри?

Репозиторий содержит файл `new-settings-2025.blk` с уже настроенным управлением. Основные изменения и улучшения:

*   **🚁 Управление вертолетами:** Добавлены и настроены удобные клавиши для управления вертолетами (актуально с коммита от 16 марта 2026 года).
*   **🔄 Готовые профили:** Вам не нужно перенастраивать каждую клавишу с нуля — просто замените файл и играйте.
*   **🎯 Оптимизация:** Подобраны эргономичные сочетания для основных действий в игре.

## 📥 Установка

1.  **Скачайте файл** `new-settings-2025.blk` из этого репозитория.
2.  **Найдите папку с настройками War Thunder.** Обычно она находится здесь:
    *   `C:\Users\[Имя_пользователя]\Documents\My Games\WarThunder\Saves`
3.  **Создайте резервную копию** вашего текущего файла настроек (например, `config.blk`), если не хотите их потерять.
4.  **Скопируйте** скачанный файл `new-settings-2025.blk` в эту папку, **переименовав его** в `config.blk` (или заменив существующий файл).
5.  Запустите игру. Новые настройки применятся автоматически.
